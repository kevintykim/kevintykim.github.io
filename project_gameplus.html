Objective: GamePlus is an online marketplace where people can shop for retro video games—a difficult product to find in physical stores today. This app is built to help you find the games you want as quickly as possible, and has been designed for phones, tablets, and desktop computers.


Role: UI Designer


Tools: Adobe XD


Duration: 1 month




Brand Guidelines

The GamePlus brand is designed to emphasize simplicity and sleekness, combining retro charm with modern convenience.



User Flow

With the brand guidelines established, the user flow diagram is conceptualized, based on several desired user tasks and functionalities.



Low/Mid-Fidelity Wireframes

The search task is visualized into a series of low-fidelity sketches. Once the sketches look good, it’s time to transform them into mid-fidelity designs by adding more detail and visual precision.



Usability Test Results

Prototype testing for the mid-fidelity wireframes brings several usability issues into perspective, setting up a smoother design process for the app’s high-fidelity wireframes.



Final Mockups

The designs are complete. The user feedback has been taken into consideration, and the brand guidelines have been fully applied. GamePlus is ready for development!
